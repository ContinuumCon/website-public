Import schedule Subscribe How to add

Add schedule to your calendar

Subscribe

Subscribe to the hosted calendar so your device receives updates automatically.

Subscription URL: /assets/continuumcon-schedule.ics

Gmail: Google Calendar → Other calendars (+) → "From URL" → paste the URL.

Outlook (web): Add calendar → Subscribe from web → paste the URL.

Import (one-time)

Download the .ics file and import it into your calendar app if you prefer a manual copy. Imported events will not receive future updates.

Download link: /assets/continuumcon-schedule.ics

  1. Outlook (desktop): File → Open & Export → Import/Export → Import an iCalendar (.ics) and follow the prompt to import into a calendar.
  2. Thunderbird (Lightning): Events and Tasks → Import, then select the downloaded .ics file and choose the calendar to add to.
